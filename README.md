- group: グループでの作業用。好きに使用してOK
- lesson_sample: 例図
- （例）05_web_ronbun = 章番号＋システム名

  - class: クラス図
  - usecase: ユースケース図
  - analysis: 分析モデル（ロバストネス図）
  - sequence: シーケンス図
